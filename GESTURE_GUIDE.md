## TROUBLESHOOTING GESTURES

### "Hand not detected"
- Check lighting
- Move closer to camera
- Extend fingers clearly
- Lower detection confidence in code

### "Gesture not recognized"
- Make movement more pronounced
- Pause between gestures
- Ensure full hand visibility
- Check gesture is supported in your version

### "Jittery/unstable tracking"
- Improve lighting
- Steady hand movements
- Increase tracking confidence
- Reduce background motion

### "Rotation too sensitive"
- Adjust rotation_speed in code:
  ```python
  self.rotation_speed = 1.0  # Lower = less sensitive
  ```

### "Scaling too slow"
- Adjust expand_threshold:
  ```python
  self.expand_threshold = 0.10  # Lower = more responsive
  ```
